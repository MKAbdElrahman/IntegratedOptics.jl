Defining Grids

In order to initialize a simulation, you have to define a grid upon which all computations will be done. Currently, Cartesian grids, with uniform spacing along each dimension are supported.

Grids are defined by using the Grid constructor which must take as input two keword arguments:

  • spacing = (dx,dy,..): the cell sizes for each dimension.
  • extent = (Lx,Ly,..): the grid length for each dimension.

spacing and extent mush have the same length.

Examples:

1D Grid
Lx = 15
dx = 0.05
Grid(extent = (Lx,) , spacing = (dx,) )
dimension:  1
size     :  (300,)
extent   :  (15.0,)
spacing  :  (0.05,)
2D Grid
Lx = 15
Ly = 7.5
dx = 0.05
dy = 0.2
Grid(extent = (Lx,Ly) , spacing = (dx,dy) )
dimension:  2
size     :  (300, 38)
extent   :  (15.0, 7.5)
spacing  :  (0.05, 0.2)
3D Grid
Lx = 15
Ly = 7.5
Lz = 4.5
dx = 0.05
dy = 0.2
dz = 0.1
Grid(extent = (Lx,Ly,Lz) , spacing = (dx,dy,dz) )
dimension:  3
size     :  (300, 38, 45)
extent   :  (15.0, 7.5, 4.5)
spacing  :  (0.05, 0.2, 0.1)
